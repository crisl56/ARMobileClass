# AR A2 Mobile Assignment

## About
This is my submission for the A2 Assignment in my mobile class.

It is a app where you can spawn a character and control them in AR with the joystick and even make them do an action move.

Pretty fun to do.

To see whats happening behind the scenes, check the scene ARWalkingCharacter.unity

## How to use

Click on the plane or press the spawn button once the plane has been made.

Control the guy with the joystick and use the A button for the action!
